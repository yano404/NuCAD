# NuCAD

NuCAD is a CAD tool focused on designing nuclear physics experimental instruments.

## Requirements

- python >= 3.10
- cadquery >= 2.5.2
- cadquery-ocp >= 7.7.2
- numpy >= 2.2.6
- pycatima >= 1.98
